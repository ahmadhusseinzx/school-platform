📚 دليل حالات العام الدراسي والفصول

مرجع سريع لإدارة العام الدراسي والفصول في نظام المنصة التعليمية

1️⃣ حالات العام الدراسي (Academic Year)

📌 الحقل: academicYear.status

الحالة المعنى ما يظهر في الواجهة الإجراءات المسموحة
active العام نشط وجاري العمل عليه ✅ نشط يمكن إدارة الفصول والعلامات
closed العام مغلق (منتهي) 🔒 مغلق لا يمكن التعديل، يظهر زر "بدء العام الدراسي"
غير موجود لم يبدأ بعد ⏳ لم يبدأ يظهر زر "بدء العام الدراسي"

2️⃣ حالات الفصول الدراسية (Semesters)

📌 الحقول: semesters.semester1.status و semesters.semester2.status

الحالة المعنى ما يظهر في الواجهة الإجراءات المسموحة
active الفصل مفتوح (يمكن التعديل) ✅ مفتوح يمكن إدخال وتعديل العلامات
closed الفصل مغلق (لا يمكن التعديل) 🔒 مغلق لا يمكن تعديل العلامات
pending في انتظار الفتح (لم يبدأ بعد) ⏳ معلق لا يمكن التعديل حتى يتم الفتح

3️⃣ تدفق العام الدراسي

🚀 بداية العام

4️⃣ مثال من Firestore

🔹 عام نشط (قيد العمل)

"academicYear": {
  "current": "2026-2027",
  "startDate": "2026-07-16T...",
  "status": "active" ✅
},
"semesters": {
  "semester1": { "status": "active" }, ✅ مفتوح
  "semester2": { "status": "pending" } ⏳ معلق
}

🔹 عام مغلق (منتهي)

"academicYear": {
  "current": "2026-2027",
  "startDate": "2026-07-16T...",
  "status": "closed" 🔒
},
"semesters": {
  "semester1": { "status": "closed" }, 🔒 مغلق
  "semester2": { "status": "closed" } 🔒 مغلق
}

5️⃣ كيفية التعديل في Firestore

📍 الخطوات:

  1. اذهب إلى Firebase Console → Firestore Database
  2. اختر مجموعة schoolSettings
  3. افتح المستند (يوجد عادةً مستند واحد)
  4. عدّل القيم حسب الجدول أدناه
  5. اضغط Save أو Update
  6. حدّث الصفحة في التطبيق (Ctrl + F5)

📋 جدول التعديلات السريعة

ما تريد فعله التعديل المطلوب
✅ إلغاء إغلاق العام academicYear.status = "active"
✅ فتح الفصل الأول semesters.semester1.status = "active"
✅ فتح الفصل الثاني semesters.semester2.status = "active"
🔒 إغلاق الفصل الأول semesters.semester1.status = "closed"
🔒 إغلاق الفصل الثاني semesters.semester2.status = "closed"
🔄 بدء عام جديد academicYear.status = "active"
semester1.status = "active"
semester2.status = "pending"

6️⃣ ⚠️ تنبيهات مهمة

🔴 القيمة الصحيحة: يجب استخدام "active" وليس "open" أو "true"
🔄 التحديث في الواجهة: بعد التعديل في Firestore، قد تحتاج إلى تحديث الصفحة (Ctrl + F5) لظهور التغييرات
📊 تأثير الحالة على العلامات:
  • عندما يكون الفصل active → يمكن تعديل العلامات ✅
  • عندما يكون الفصل closed → لا يمكن تعديل العلامات ❌

7️⃣ 📋 ملخص سريع

العنصر مفتوح/نشط مغلق/منتهي معلق/لم يبدأ
العام الدراسي active closed (غير موجود)
الفصل الأول active closed pending
الفصل الثاني active closed pending

8️⃣ 🎯 الأزرار في الواجهة

الزر شروط الظهور الوظيفة
▶️ بدء العام الدراسي academicYear.status !== "active" بدء عام دراسي جديد
🔒 إغلاق الفصل الأول selectedSemester === 1 && semester1.status === "active" إغلاق الفصل الأول ومنع التعديل
🔒 إغلاق الفصل الثاني selectedSemester === 2 && semester2.status === "active" إغلاق الفصل الثاني ومنع التعديل
🔓 فتح الفصل الأول selectedSemester === 1 && semester1.status === "closed" فتح الفصل الأول للسماح بالتعديل
🔓 فتح الفصل الثاني selectedSemester === 2 && semester2.status === "closed" فتح الفصل الثاني للسماح بالتعديل
🚩 إغلاق العام الدراسي semester1.status === "closed" && semester2.status === "closed" إغلاق العام الدراسي بالكامل

9️⃣ 💡 مثال عملي

🔹 السيناريو: أغلقت العام الدراسي 2026-2027 بالخطأ

المشكلة: العام الدراسي مغلق ولا يمكن تعديل العلامات

الحل:

  1. اذهب إلى Firestore → schoolSettings
  2. غيّر academicYear.status من "closed" إلى "active"
  3. إذا كنت تريد فتح الفصلين أيضاً، غيّر:
    • semesters.semester1.status → "active"
    • semesters.semester2.status → "active"
  4. احفظ التغييرات وحدّث الصفحة
  5. ✅ ستعود العلامات قابلة للتعديل
